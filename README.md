# Driver Drowsiness Detection System

This project detects driver fatigue using a webcam and alerts the driver with an alarm when drowsiness is detected.

## Technologies Used
- Python
- OpenCV
- Computer Vision
- Haar Cascade Classifier

## Features
- Real-time face detection
- Eye detection
- Alarm alert when driver closes eyes for too long

## How to Run

Install dependencies:

pip install opencv-python numpy

Run the project:

python detect_drowsiness.py
